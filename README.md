# SEAL  
**Semi-automated Ecological Annotation Logger**

SEAL is a lightweight frame-based annotation tool for ecological time-lapse and survey imagery.

It supports:
- Manual point marking
- Event logging via hotkeys
- Optional template-based tracking
- MARK & TRACK mode (continuous surveys)
- MARK & VIEW mode (CEMP / sparse sampling)
